# 首次试用示例任务

这些示例只用于帮助新用户理解提问方式；不会自动写入知识库、长期记忆或项目空间。

1. `帮我把今天的工作整理成待办清单，并按优先级排序。`
2. `请为“下周上线一个新功能”制定项目计划，列出里程碑、风险和下一步。`
3. `请总结下面这段内容，并给出三个可执行的下一步：……`
4. 上传一份非敏感的 Markdown 或 TXT 示例资料后：`根据我上传的资料，提炼核心结论并标明依据。`

如需测试图片资料，请上传不含敏感信息的清晰截图；如本机没有 OCR，平台会提示安装方式，不会将图片发送到外部服务。

## 使用边界

- 不要将演示资料当作正式知识库内容；测试完成后可在“知识库”中删除。
- 不要上传密码、访问令牌、客户隐私或无权处理的文件。
- 需要验证文件生成时，先要求生成 Markdown；确认卡出现后再决定是否创建本机文件。
