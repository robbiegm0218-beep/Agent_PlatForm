# Agent_Platform 当前执行计划

更新日期：2026-07-23

本文件是唯一的当前任务入口。已完成阶段的详细记录保存在 [历史实施计划](./implementation-plan-history.md)，操作说明保存在 [运维手册](./operations-guide.md) 和 [运维 Runbook](./operations-runbook.md)。

## 1. 当前目标

交付可由个人长期托管、可邀请少量测试用户使用的 Agent 平台测试版。

当前阶段不扩展到正式商业 SaaS，不实现团队计费、复杂组织权限、企业 SSO、多区域部署和大规模运营。先确保单实例在安全、数据、恢复、成本和体验方面可控。

## 2. 状态口径

- `[x]`：开发与必要验证已完成。
- `[~]`：正在执行，尚未满足全部验收条件。
- `[ ]`：已规划但尚未开始。
- `[-]`：当前阶段明确推迟，不阻塞个人托管测试版。
- `阻塞发布`：不影响本地开发使用，但未处理前不能开放给外部测试用户。
